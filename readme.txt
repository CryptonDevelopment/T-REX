Установка и распаковка
-Скачайте архив с программой.
-Распакуйте его в отдельную папку.
Важно: Не запускайте программу прямо из архива.

Установка библиотек
-Откройте папку с программой.
-Нажмите правой кнопкой мыши на пустое место -> "Открыть в терминале".
-Вставьте эту команду и нажмите Enter:
 pip install -r requirements.txt  
-Затем установите браузерный движок:
 playwright install chromium 
-Выполните команду python main.py

Перед стартом
Перед запуском автоматизации необходимо выполнить несколько действий вручную, чтобы софт работал корректно. 	

Убедитесь, что у вас запущен AdsPower.

Ручной вход
Софт не умеет вводить логин и пароль (это сделано специально для безопасности, чтобы не хранить пароли).
-Откройте каждый профиль в AdsPower вручную.
-Зайдите на сайт (https://www.trex.xyz/portal).
-Авторизуйтесь
Теперь браузер запомнил вашу сессию, и софт сможет сразу приступить к работе.

Функционал:
Импорт аккаунтов
Формат файла (каждый аккаунт с новой строки):
j94k201 
k102934:MainAccount 
Можно указывать просто ID, а можно id:name для удобства.

Настройка потоков
Ползунок CONCURRENCY.
1 Thread: Делает аккаунты по очереди.
2-10 Threads: Делает 2-10 аккаунтов одновременно.
Старт:
Нажмите SELECT ALL (или выберите нужные аккаунты вручную).
Нажмите кнопку INITIALIZE_SEQUENCE.

Кнопка TERMINATE останавливает работу. Софт доделает текущее действие, закроет браузеры и только потом остановится.
