T-REX Quest Bot
===============

О ПРОГРАММЕ
-----------
Скрипт собирает дейлики проекта на портале (https://www.trex.xyz/portal/quest).
Основная функция — автоматизация "Daily Check-in".

УСТАНОВКА И ЗАПУСК
------------------
1. Установите Python (если не установлен).
2. Установите зависимости:
   pip install -r requirements.txt
3. Установите браузеры Playwright:
   playwright install
4. Запустите программу:
   python gui.py

ИНСТРУКЦИЯ ПО ИСПОЛЬЗОВАНИЮ
---------------------------
1. Импорт профилей:
   - Нажмите "IMPORT" в интерфейсе программы.
   - Выберите файл с профилями Ads Power.
   - Формат строк в файле: id profile:profile_name

2. Настройка потоков:
   - Выберите количество одновременно работающих профилей (от 1 до 10) с помощью ползунка.

3. Управление:
   - Запуск: Нажмите "INITIALIZE_SEQUENCE".
   - Остановка: В случае неисправности нажмите "TERMINATE" для экстренного завершения.

ВАЖНОЕ ПРИМЕЧАНИЕ!
------------------
Перед работой скрипта ОБЯЗАТЕЛЬНО зайдите на портал https://www.trex.xyz/portal/quest и залогиньтесь вручную во всех профилях.
После этого можно запускать скрипт.

ТЕКУЩИЙ СТАТУС РАЗРАБОТКИ
-------------------------
Реализована работа кнопки "Daily check-in".
Другие квесты требуют действий пользователя (расширения, соцсети, бейджи), поэтому пока не автоматизированы.
Планируется добавление новых заданий, если будет большое количество пользователей с NFT или балансом.
